Jetty 6 JNDI and Transactions

This example shows how to configure and lookup DataSources and a JTA transaction manager in JNDI, and how to perform a distributed transaction across two different databases.

Preparation

Make sure you have built this demo. Assuming your jetty src distro is installed at $JETTY_HOME: 

  >  cd $JETTY_HOME/modules/test-jndi-webapp
  >  mvn -Dtxmgr=xxxx clean install
  >  cd $JETTY_HOME
  > java -jar start.jar etc/jetty.xml etc/jetty-plus.xml etc/jetty-test-jndi.xml

Where xxxx is the name of a transaction manager to use. There is a list of these that are supported for the demo in $JETTY_HOME/modules/test-jndi-webapp/src/templates. Ensure that you have installed the jars etc for the transaction manager of your choice in jetty6.
