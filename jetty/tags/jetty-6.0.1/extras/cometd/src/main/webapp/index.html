Cometd demo

To run this demo you need to do the following checkouts into this directory. 

  svn co http://svn.dojotoolkit.org/dojo/trunk dojo
  svn co http://svn.xantus.org/shortbus/trunk/cometd-twisted/tests tests

Then hit the magnet demo with two different browsers and drag magnets to see the browsers communicate via cometd

Or try the the chat demo.
