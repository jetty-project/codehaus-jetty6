Welcome to Jetty Hightide 7

The Jetty project is a 100% Java HTTP Server, HTTP Client and Servlet Container. The core project is hosted by the Eclipse Foundation at http://www.eclipse.org/jetty. The jetty integrations with 3rd party modules are hosted by Codehaus at http://jetty.codehaus.org.

This is the Jetty Hightide distribution which contains the core jetty modules, plus the 3rd party dependencies and integrations needed to create a full-featured application server.
